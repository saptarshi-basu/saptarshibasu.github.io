    Transpiling ES6 into browser compatible code

    ES6
    Babel
    Browserify
    Grunt
    Transpilation

    ES6 is currently not supported by all browsers. A Grunt task can be used to transpile ES6 into browser compatible ES5 using Babel and Browserify.

    Babel transpiles next generation java script code (written in ES6/ES7) to older version javascript (ES5) which is suported by all browsers. Thus it allows us to write next generation javascript syntax and functions today even in the absence of complete browser support for these new language features.

    However, Babel uses the require() method call to support this modular approach. The require() function call comes from the CommonJs module specification which is implemented by NodeJs. Thus the Babel output is supported by NodeJs, but not by the browsers which don't implement CommonJs specification. Browserify, comes to our aid here - it combines all the module files to one single file and translates the require() calls to browser compatible code.
