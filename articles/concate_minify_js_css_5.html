    Concatenating and minifying javascript and CSS files

    CSS

    Uglify
    cssnano

    grunt-contrib-concat

    It is a good practice to concatenate and minify all javascript files into a single file for performance reasons. Same is true for CSS.

    An HTML page may depend on multiple javascript files and css files. Each file is downloaded in the user's system using a separate HTTP call. Hence, if all the dependent javascript files are concatenated and minified to form a single file and if the same is also done for CSS, only two HTTP requests will be sufficient - one for javascript and one for CSS. This is expected to reduce the overall load time of the web page.
