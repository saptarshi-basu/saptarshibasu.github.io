How to run ES6 code on browser using Babel, Grunt and Browserify

ES6 or ECMAScript 2015, released in June 2015, brings in many amazing features and syntax in javascript. Not all of these new features are currently supported by today's web browsers. However, even in the absence of complete browser support, it is a good idea to start writing javascript using the ES6/ES7 features.

There are two reasons for this:

  1. ES6/ES7 syntax promotes cleaner and maintainable code
  2. The support for ES6/ES7 in the browsers is just a matter of time

Therefore we can use a transpiler like Babel to translate the ES6 syntax to ES5 syntax which is supported by the browsers. In this article, I'll show how to do that. I'll use the following tools for this purpose:

  1. npm - It helps in managing the project dependencies.
  2. Grunt - It is a task runner that automates the repetitive tasks such as transpiling java script, minifying javascript, minifying css etc.
  3. Babel - It transpiles next generation java script code (written in ES6/ES7) to older version javascript (ES5) which is suported by all browsers. Thus it allows us to write next generation javascript syntax and functions today even in the absence of complete browser support for these new language features.
  4. Browserify - The Babel output doesn't run in browsers directly. The Babel output is still moduler - there is a one-to-one mapping between our source module files with the output files created by Babel. However, Babel uses the require() method call to support this modular approach. The require() function call comes from the CommonJs module specification which is implemented by NodeJs. Thus the Babel output is supported by NodeJs, but not by the browsers which don't implement CommonJs specification. Browserify, comes to our aid here - it combines all the module files to one single file and translates the require() calls to browser compatible code.

With this much background, we'll now get our hands dirty and do a small project.

Step 1 Install NodeJs. This will automatically install npm.

Step 2 Create a project folder "testproject" and change to that diretory.

Step 3 Go to the command line and execute the following commands:

npm init

npm install grunt --save-dev

npm install grunt-babel --save-dev

npm install babel-preset-es2015 --save-dev

npm install grunt-browserify --save-dev

Step 4 Create the javascript file foo.js at the location testproject/src/js/

Step 5 Create the javascript file bar.js at location testproject/src/js/

Step 6 Create the html file index.html at location testproject/

Step 7 Create the grunt file Gruntfile.html at location testproject/

Step 8 Build the project by running the following command at location testproject/

grunt build
